# 제품 요약 PRD (v1)

## 1) 제품 한 줄

**내 사진 업로드 → 연령대/라이프스테이지 + 장르 선택 → ‘내가 주인공’인 오리지널 세계관 영상 생성**

* 기본은 바이럴 목적(개인 엔터테인먼트)
* 옵션: **Rewrite Moment(잊지 못하는 강력한 사건)**을 “유사 상황”으로 재구성해 **다른 결말**로 바꾼 엔딩 제공

## 2) 핵심 가치

* “나” 중심 몰입 + 바로 공유 가능한 숏폼 결과물
* 장르/스테이지/결말 조합으로 반복 생성 욕구
* **저작권/IP 문제 없는 오리지널 세계관**으로 안정적인 상품화

---

# 2) 입력/옵션

## 필수 입력

1. **내 사진 1~3장 업로드**(정면/측면/웃는 표정 권장)
2. **스테이지(연령대/라이프 스테이지) 선택**

   * MVP 1차: **청소년 / 20대 / 신혼 / 초기 육아**
   * 확장: 유년, 30대, 40대+, 유아육아/사춘기자녀 등
3. **장르 선택(영화 장르처럼)**

   * MVP 1차: **다큐 / 코미디 / 드라마 / 멜로 / 판타지**

## 선택 입력: Rewrite Moment (핵심 기능)

* 사건 한 줄(최대 300자)
* **거리두기**: 상징적으로 / 유사 상황 / 꽤 비슷하게
* **바꿀 결말**: 회복 / 성장 / 화해 / 자기보호 / 새출발 / 코미디 승화
* 인물 처리 기본값: **실명·실존인물 금지, 익명/실루엣 처리**

## 공통 슬라이더 3개

* 현실감(현실↔영화)
* 강도(순한맛↔진한맛) *폭력/공포 수위는 MVP에서 제한*
* 속도(느림↔빠름)

---

# 3) 출력(길이 3단) + 무료/유료 분리

## A. Quick Pack (무료 중심)

* **8초 × 3개 클립** (Hook / Main / Ending)
* 포맷: 9:16 우선
* 기본 자막 + 워터마크(서비스명 + Fiction/Simulation)

## B. Story Pack (유료)

* **15초 × 3개 클립**
* 한 클립에 “상황→감정→전환”이 들어가도록 비트(beat) 2~3개 포함

## C. Trailer (유료 프리미엄)

* **45~60초 × 1개**
* 6~10장면으로 구성된 “예고편” 형태
* 내레이션/타이틀 카드/엔딩 카드 포함

---

# 4) Rewrite Moment 적용 규칙(재구성)

## Quick/Story

* **Clip 3(Ending)** 을 Rewrite 버전으로 교체

## Trailer

* 2막 중간에 **Rewrite 장면 1개 삽입** + 엔딩을 선택 결말로 변경
* “Rewrite 장면만 재생성” 버튼 제공(비용/시간 절약)

---

# 5) 저작권/안전 정책(운영 필수)

## 저작권(IP) 안전 모드

* 금지: 영화/드라마/만화 **작품명·캐릭터명·특정 고유 상징·로고·유명 대사**
* 감지 시: 장르 키워드로 자동 치환(“마법학교풍/근미래SF/도시느와르(오리지널)” 등)
* 결과물은 **오리지널 세계관(가상 도시/가상 단체/가상 헤드라인)**만 사용

## Rewrite Moment 안전 모드

* “직접 재현” 금지: 실명/학교/회사/주소/정확한 날짜/식별 가능한 묘사 제거
* 민감 주제(자해/학대/극단적 폭력 등) 강하면:

  * 상징화/완화 + 안전 안내(도움 요청) + 직접 재현 요청은 거절
* 가족/배우자/아이 얼굴은 기본 **비디테일(실루엣/손/뒷모습/얕은 심도)**

## 표시 의무

* 결과물 워터마크/고지: **Fiction / Simulation(가상의 창작물)**

---

# 6) 화면 구성(4 screens)

1. **업로드**: 사진 업로드 + 동의 체크
2. **선택**: 스테이지/장르 + 슬라이더 + 길이 모드 선택
3. **Rewrite Moment(선택)**: 사건 입력/거리두기/결말
4. **결과**: 클립 재생/다운로드/공유 + 재생성(클립/장면 단위) + 업그레이드 CTA

---

# 7) 생성 파이프라인(기술 개요)

1. 입력 정제: IP/개인정보 마스킹
2. **Identity Pack 생성**: 선택한 스테이지 버전 “기준 얼굴 이미지 세트” 생성
3. 스토리보드(JSON) 생성
4. 키프레임 이미지 생성(장면별)
5. 영상 생성(8초×3 / 15초×3 / 60초 트레일러)
6. 서버 합성: 자막/워터마크/타이틀 카드/붙이기
7. 자동 검수: 깨짐/금칙어/과도한 수위 → 문제 구간만 재생성(최대 2회)

---

# 8) 크레딧/요금 정책(확정)

## 무료 정책 (Free)

* **Quick Pack(8초×3)만 가능**
* 추천 제한: **하루 1팩**(또는 주 3팩)
* 워터마크 강/중, 720p, 생성 큐 우선순위 낮음
* Rewrite Moment:

  * 옵션 A(추천): **주 1회만 허용**
  * 옵션 B: 무료는 **Rewrite 미리보기(스토리보드)까지만**
    *(둘 중 하나 택하면 됨 — 운영 난이도 기준으로 A가 단순)*

## 유료 정책 (Paid: 크레딧 소모)

* Story Pack(15초×3) = **2 크레딧**
* Trailer(45~60초×1) = **5 크레딧**
* Rewrite Moment 추가 = **+1 크레딧**
* 재생성

  * Story: 클립 1개 재생성 = **0.5~1 크레딧**
  * Trailer: 장면 1개 재생성 = **1 크레딧**
* 유료 혜택: 1080p, 빠른 큐, 워터마크 약하게/옵션, 클립/장면 단위 재생성

## 전환 UX(무료→유료)

* 결과 화면 CTA:

  * **[15초로 업그레이드]**
  * **[60초 트레일러 만들기]**
* 무료 Quick 결과를 본 직후 “Story Pack은 전환/반전/내레이션이 포함” 같은 차이점 미리보기 제공

---

# 9) MVP 1차 범위(2주용)

* 스테이지 4개: 청소년/20대/신혼/초기육아
* 장르 5개: 다큐/코미디/드라마/멜로/판타지
* 무료: Quick Pack(8초×3)
* 유료: Story(15×3), Trailer(60×1)
* Rewrite Moment: Clip3(엔딩) 교체(Quick/Story), Trailer는 장면 1개+엔딩 분기
* 공유/다운로드/재생성 정책 포함
